Kalyptica – News Site

Full-stack global news platform built with Next.js, Node.js & SQL

Project Overview

Kalyptica is a modern news platform designed to deliver news from around the globe through a clean and accessible interface. The project consists of a Next.js frontend for readers and a dedicated backend with an admin panel for managing news content.

The system follows a clear separation of concerns, with a frontend web application for users and a Node.js-powered backend along with an admin dashboard for content management.

Key Features

  • Global news aggregation and presentation
  • Clean and responsive user interface
  • Dedicated admin panel for managing news content
  • Backend API for handling news data
  • Live deployed frontend, backend, and admin panel
  • Scalable architecture with frontend-backend separation

Tech Stack

Next.js Node.js EJS SQL Database JavaScript HTML CSS

Frontend

The frontend is built using Next.js and provides a fast, SEO-friendly, and user-focused experience. It consumes data from the backend server through configured API endpoints and is deployed live for public access.

Backend & Admin Panel

The backend is developed using Node.js, with an admin panel built using EJS templating. The admin panel allows authorized users to manage and publish news content efficiently, while the server handles data operations and API requests.
