NoteInk – AI Powered Note-Taking App

Cross-platform AI-powered note-taking application built with Next.js, Expo & Gemini AI

Project Overview

NoteInk is a modern AI-powered note-taking application designed to enhance productivity through intelligent assistance. The app provides a clean and intuitive interface while integrating an AI assistant that helps users summarize content, answer queries, and organize notes efficiently.

The project follows a monorepo architecture and supports both web and mobile platforms, ensuring a seamless experience across devices.

Key Features

  • AI assistant for answering queries and summarizing notes
  • Cross-platform support for web and mobile applications
  • Clean, simple, and user-friendly interface
  • Fast and developer-friendly project setup
  • Monorepo structure for scalable development

Tech Stack

Next.js Expo React Native Gemini AI JavaScript HTML CSS

AI Integration

NoteInk integrates Google’s Gemini AI to provide intelligent note assistance. The AI helps users interact with their notes by generating summaries, answering questions, and improving content organization through contextual understanding.

Cross-Platform Architecture

The application is structured as a monorepo containing two separate apps: a mobile application built with Expo and a web application built with Next.js. This architecture ensures code consistency while allowing platform-specific optimizations.

Contribution & Collaboration

NoteInk is an open-source project and welcomes community contributions. Developers can contribute by improving features, enhancing UI/UX, fixing bugs, or improving documentation through GitHub pull requests.
